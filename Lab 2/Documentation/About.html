About the project

Augmented Reality: Travel and Tourism

Designing an application that uses Augmented Reality to aid in travel and tourism.

The main functionality of the application is to enable the user to translate their preferred language to the local language of the region.

Additional functionality of having Google Maps built in for the user to navigate will be added

Speech to Text, Speech to Speech, Text to Speech and Text to Text translation will be made available in later releases
